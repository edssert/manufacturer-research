# 업데이트 기록

이 문서는 이 프로젝트(음향 장비 비교 사이트)가 그동안 어떻게 좋아져 왔는지
누구나 이해할 수 있게 정리한 기록입니다. 최신 내용이 맨 위에 있습니다.

개발자용 상세 기록(파일명, 코드 구조 등)은 같은 폴더의
`CHANGELOG-dev-detailed.md`에 있습니다.

---

## v1.8 (2026-07-15) — 화면 상태 그대로 주소 공유, 더 똑똑한 검색, 모바일 개선

- **주소(URL) 공유**: 제품 상세 팝업, 나란히 보기(두 제품 비교), 사진
  확대 화면까지 지금 보고 있는 상태 그대로 주소창에 반영됩니다. 주소를
  복사해 보내면 받은 사람도 같은 화면이 바로 열리고, 브라우저
  뒤로가기를 누르면 팝업이 닫힙니다.
- **더 똑똑한 검색**: 띄어쓰기나 하이픈이 달라도 같은 제품을 찾아줍니다
  (예: "KS 28", "ks-28", "KS28" 모두 동일). 회사 이름도 "lacoustics"
  처럼 붙여 써도 검색됩니다. 빠르게 타이핑할 때 화면이 버벅이지 않도록
  처리도 개선했습니다.
- **모바일 개선**: 좁은 화면에서 뭉개지던 상세 표를 옆으로 넘겨 볼 수
  있게 했고, 크기/무게/방수등급 표를 두 줄로 재배치했습니다.
- **더 빠른 로딩**: 사진을 실제로 보이는 순간에만 내려받도록 하고,
  글꼴을 먼저 불러와 첫 화면 깜빡임을 줄였습니다.
- **내부 정리**: 코드 구조를 더 다루기 쉽게 나누고(팝업 코드 분리),
  자동 검사(테스트)를 현재 상태에 맞게 손봐 전부 통과하도록 했습니다.
  예전 폴더(`raw-data/legacy`)는 문서를 `docs/`로 옮기고 삭제했습니다.

## v1.7 (2026-07-14) — K1 상세 화면에 Mechanical Safety(기계적 안전) 섹션 추가

K1 스피커 상세 화면에 공식 오너 매뉴얼의 "Mechanical safety"(기계적
안전) 내용을 새 섹션으로 추가했습니다. Preset Guide와 같은 등급의
섹션으로, 접어둔 상태로 시작해 필요할 때만 펼쳐 볼 수 있습니다.

- K1을 매달아(flown) 설치할 때, 그리고 쌓아(stacked) 설치할 때 각각
  안전하게 사용할 수 있는 최대 대수를 표로 정리했습니다.
- 풀백 금지 등 반드시 지켜야 할 경고 사항을 눈에 띄게 표시했습니다.
- 안전계수 산정 원칙, Soundvision 활용법 등 상세 설명은 참고 사항
  토글 안에 모아뒀습니다.
- K1-SB·KS28·CS1 등 다른 제품의 안전 한계 표는 이번에는 포함하지
  않았습니다 — 각 제품 자체 페이지에 정리할 예정입니다.

## v1.7 (2026-07-14) — 사진 확대 후 배경 클릭 시 나란히 보기가 통째로 꺼지던 오류 수정

나란히 보기(Split View)에서 오른쪽에 다른 제품을 열어둔 상태로 그
제품의 사진을 확대해서 보다가, 화면 좌우 빈 공간을 클릭해 확대를 닫으면
원래 열려 있던 오른쪽 제품 화면으로 돌아가야 하는데 나란히 보기 자체가
꺼져버리던 오류를 고쳤습니다. 오른쪽 위 X 버튼으로 닫을 때는 정상
동작했지만, 배경을 클릭해서 닫는 경로만 이 복원 과정을 빠뜨리고
있었습니다. 이어서 비슷한 원인의 문제 두 가지를 추가로 고쳤습니다: 복원된 오른쪽
제품 화면에서 사진 영역만 빈 채로 보이던 것, 그리고 사진 확대 상태에서
왼쪽 제품의 다른 항목(예: 매칭 앰프)을 눌러 오른쪽 화면을 새 제품으로
바꿨을 때 왼쪽 제품의 사진이 돌아오지 않던 것입니다. 두 경우 모두 사진을
확대할 때 원본 사진 영역에 걸어두는 "접히는" 효과가 각각 다른 경로에서
풀리지 않고 남아있던 게 원인이었습니다.

## v1.7 (2026-07-14) — K1 프리셋 딜레이표 극성 오류 정정

K1 상세 화면 Preset Guide의 Delay Defaults(딜레이 기본값) 표에서, 서브
우퍼(KS28·CS1)와 노이즈 컨트롤용 K1-SB의 극성이 실제로는 반전(−)인데
전부 정상(+)으로 잘못 표시되던 오류를 바로잡았습니다. 원본 매뉴얼 PDF의
극성 아이콘이 이전 자동 추출 과정에서 텍스트로 인식되지 않아 누락됐던
것으로, 이번에 페이지를 이미지로 다시 확인해 정정했습니다.

- K1은 모든 조합에서 항상 정상(+)이며, K1-SB는 대부분 정상이지만
  노이즈 컨트롤용 프리셋([K1SB_100_NC])에서는 반전입니다.
- KS28과 CS1은 K1과 함께 쓰는 모든 조합에서 항상 반전(−)입니다 — 이건
  의도된 설정(카디오이드 패턴을 만들기 위함)입니다.
- 표에서는 반전(−)인 항목만 빨간 글씨로 표시해 한눈에 구분되게
  했습니다.
- 값이 여러 개 이어지는 칸(예: K1/K1-SB/KS28 값이 한 칸에 같이 표시되는
  경우)은 텍스트로 구분하는 대신 표 안에 짧은 구분선을 그어 항목별로
  더 뚜렷하게 나눠 보이게 했습니다.

## v1.7 (2026-07-14) — FHD 화면에서 스피커 확대 사진 하단 잘림 수정 (검증 완료)

풀HD(FHD) 모니터로 크롬을 전체화면으로 켰을 때, 화면 배율이 100%인
경우에 한해 K1/K1-SB처럼 세로로 긴 Array(여러 대를 쌓은 모습) 사진을
확대해서 볼 때 사진 아래쪽이 잘려 보이던 문제를 고쳤습니다. 사진을
표시하는 화면 요소가 특정 화면 크기·배율 조합에서 실제 필요한 크기를
잘못 계산하던 게 원인으로 보여 관련 스타일을 수정했습니다. 실제 FHD
화면에서 더 이상 잘리지 않는 것을 확인했고, 사진 아래 여백이 거의 없어
보이던 부분도 미세하게 여백이 남도록 다듬었습니다.

- 사용자가 실제 FHD 화면에서 직접 확인해 잘림이 해결됐음을 확인해
  주었습니다.

## v1.7 (2026-07-14) — K1 프리셋 가이드 심층 보강 (오너 매뉴얼 반영)

K1 스피커 상세 화면의 Preset Guide 섹션에 공식 오너 매뉴얼(K1 owner's
manual EN v4.0)의 내용을 추가로 반영했습니다.

  필요한지) 정보를 새로 추가했습니다.
- 프리셋 조합별 딜레이·극성 기본값을 별도 표(Delay Defaults)로 정리해
  Preset Guide 하위에서 펼쳐볼 수 있게 했습니다.
- Kara II·K2를 다운필(아래쪽 관객을 보완하는 추가 스피커)로 쓸 때의
  프리셋 구성도 표에 추가했습니다.
- 매뉴얼 전체 목차를 요약해 raw-specs 원문 파일에 기록해 두었습니다.

## v1.7 (2026-07-13)

제품 상세 화면(모달)과 나란히 비교 화면(Split View)을 다듬은 작업입니다.
새로 추가한 프리셋 가이드 정보와 함께, 그동안 불편했던 스크롤/화면 흔들림
문제들을 정리했습니다.

- L-Acoustics K1 스피커의 상세 화면에 제조사 프리셋 구성 가이드
  ("Preset Guide") 섹션을 새로 추가했습니다. 스피커 구성별로 어떤 프리셋을
  써야 하는지, 그때 음향 특성이 어떻게 달라지는지 표로 볼 수 있습니다.
  기본은 접힌 상태로 시작하고, 눌러서 펼쳐볼 수 있습니다.
- 나란히 비교(Split View) 기능을 다시 다듬었습니다. 이제 두 제품이 각각
  독립된 카드처럼(그림자·테두리·둥근모서리 모두 따로) 나란히 보이고,
  스크롤도 서로 완전히 따로 움직입니다. 처음 카드 하나만 열었을 때도
  나중에 비교 화면으로 넓어질 자리에 미리 자리 잡고 있어, 비교 화면으로
  전환돼도 왼쪽 카드가 움직이지 않습니다.
- 상세 화면과 비교 화면의 좌우 여백을 제품 목록 화면(카드 4개씩 보이는
  그리드)의 여백과 정확히 맞췄습니다.
- 상세 화면을 스크롤해도 제품명·태그가 있는 맨 위 영역은 항상 화면에
  고정되도록 했습니다.
- 스크롤바를 더 얇고 세련되게 바꿨습니다 — 평소엔 안 보이다가 스크롤하는
  동안이나 마우스를 올렸을 때만 짧고 얇은 막대가 나타나며, 카드의 둥근
  모서리를 침범하지 않습니다.
- 상세 화면을 닫았다가 다시 열면 항상 맨 위에서 시작하도록 고쳤습니다
  (예전엔 이전에 보던 위치가 그대로 남아있던 경우가 있었습니다).
- 나란히 비교 화면을 열고 닫을 때 왼쪽 화면의 스크롤 위치가 갑자기
  맨 위로 튀거나 화면이 위아래로 살짝 흔들리던 문제를 고쳤습니다.
- 상세 화면(또는 비교 화면)이 열려 있는 동안에는 뒤에 있는 제품 목록
  화면이 스크롤되지 않도록 완전히 잠갔습니다.
- 상세 화면 속 사진 위 여백이 단일 화면과 비교 화면에서 서로 다르게
  보이던 문제를 고쳤습니다.
- 프리셋 가이드 표의 열 너비·글자 크기를 다듬어 줄바꿈 없이 한 줄로
  보이도록 최적화했습니다. Acoustic Properties 항목은 짧게 요약해
  보여주고, 마우스를 올리면 전체 내용을 팝오버로 볼 수 있게 했습니다.
  (※ 이 팝오버가 표시되지 않는 문제가 아직 남아있습니다 — 다음 세션
  이어서 확인 필요.)
- 프리셋 가이드의 참고사항(Notes) 4가지를 글머리 기호 목록으로 정리해
  가독성을 높였습니다.
- 액세서리끼리도(예: K1-BUMP ↔ K-BUMPFLIGHT, K1-CHARIOT2 ↔
  K1-CHARIOTCOV) 서로 연동되어, 상세 화면에서 클릭하면 나란히
  비교해볼 수 있도록 했습니다.
- 액세서리 상세 화면에 "General"이라는 섹션 제목을 추가해 기본 정보
  (Type/Price/Weight 등)를 다른 섹션과 명확히 구분했습니다. 이어서
  이 섹션이 항상 같은 위치에 오도록 화면 구성 순서도 정리했습니다.
