# SB18 (L-Acoustics) — Master Schema

**Category**: speakers | **Brand**: L-Acoustics (LA) | **Schema Phase**: Phase 3-1(동일 브랜드 이기종 서브우퍼 추가 — Kara II/K1/K2/K3 등의 delay_defaults가 이미 참조하던 "SB18"의 실제 원본 최초 투입)

**스켈레톤 근거**: `speakers/LA/KS28_v1.5.md`를 뼈대로 사용 — SB18은 KS28과 동일한 LF 전용(Way_Count=1) 패시브 서브우퍼 아키텍처이며 섹션 구성이 동일하다. 다만 KS28과 달리 (1) 단일 18" 드라이버(KS28은 2x18"), (2) 8 Ohm 공칭 임피던스(KS28은 4 Ohm), (3) IN 전용이 아닌 IN+LINK 양쪽 4-point speakON 커넥터 구조(KS28은 IN 하나만 스펙에 명시)를 가진다.

**"SB18/SB18i/SB18m" 명칭 범위 확정(원본 재검토 결과)**: 업로드된 `SB18_UM_EN.pdf`(버전 6.0)는 표지에 "SB18 (i/m)"이라 표기되어 있으나, 본문에 명시적 조항이 있다 — "In this document, the SB18 term and illustration will refer equally to SB18, SB18i or SB18m. These products are different versions of the same subwoofer... The rigging system of each version has been designed to be compatible with a distinct main system: **SB18 with Kara (II)**, SB18i with Karai, and SB18m with ARCS Focus/Wide." 즉 이 UM 하나가 SB18/SB18i/SB18m **3개 베리언트**를 함께 다루며, 전기음향 스펙(주파수 특성, Max SPL, 임피던스, 커넥터)은 3개 베리언트가 동일하고 물리(무게/리깅 시스템)만 다르다(Appendix C에 SB18/SB18i/SB18m 3개 스펙 표가 각각 별도 수록됨: SB18=52kg 통합 리깅, SB18i=52kg 통합 리깅+커스텀 RAL, SB18m=62kg 캡티브 커플링 바). 이번 작업 지시 대상인 "SB18"은 이 3자 중 **Kara (II)와 짝을 이루는 투어용 표준 버전**이며, 이는 `Kara_II_v1.6.md`의 delay_defaults/preset_guide_and_matching이 "SB18"(접미사 없음)을 참조하는 것과 정확히 일치한다 — 따라서 이 파일은 UM Appendix C의 "SB18" 표(SB18i/SB18m 표가 아님)를 채택했다. **SB18i(소문자 i, "Karai"와 짝)와 별도 문서로 제공된 SB18_IIi(대문자 II, Kara IIi와 짝)는 서로 다른 제품이다** — SB18i는 이 UM 안에서만 존재하는 물리 베리언트(52kg, speakON, 동일 스펙)이고, SB18_IIi는 완전히 별도의 OM(SB18_IIi_OM_EN_3.0.pdf, terminal block 커넥터, 48kg, IP55)을 가진 신세대 install 전용 제품이다 — 이름의 "II"가 세대 구분을 나타낸다(Kara II -> Kara IIi 계보와 동일한 논리).
