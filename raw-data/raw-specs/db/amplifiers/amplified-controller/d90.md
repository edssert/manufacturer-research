# D90 (d&b audiotechnik, Amplified Controller)

- Type: Mobile specific four channel amplifier
- Channels: 4
- Power class: Class D
- Connectivity: Milan-AVB, AES3, analog
- Rack: 2U, Weight: 18.8 kg / 41.5 lb
- Mains: 100-127 / 208-240V, 50-60Hz, powerCON®-HC
- Power supply: Autosensing SMPS with active PFC
- I/O: Analog 4ch(INPUT ANALOG A1-A4, 3pin XLR female), AES3 4ch(INPUT DIGITAL D1/2 D3/4, 3pin XLR female), Milan 8ch(INPUT Network Audio M1-M8, etherCON built-in 5-port AVB switch 100/1000Mbit)
- Network redundancy: Milan-enabled
- Connections: 2x etherCON, ANALOG LINK(3pin XLR male), DIGITAL LINK Output(3pin XLR male)
- Output: 4ch, Mix TOP/SUB / 2-Way Active A/B·C/D: 2x NL4, 4Ch output: 1x NL8, Speaker outputs A/B/C/D: 4x NL4, min 4Ω
  - 4Ω: 4x5400W / 8Ω: 4x2700W
  - Max voltage 210 Vpeak, max current 78 Apeak
- DSP: AES in 44.1/48/96/192kHz, EQ 2x16-band PEQ/notch/shelving/asymmetric
  - Preset library: Current d&b loudspeakers(ArrayCalc)
  - Latency: 0.3ms, delay 0.3ms-10s
  - Dynamic range: 119dB(unweighted)
- Ecosystem: d&b ArrayCalc simulation software, R1 Remote control software
- Features: AutoStandby/AutoWakeup, Function switches(d&b loudspeaker specific circuits), Frequency generator(Pink noise/Sine 10Hz-20kHz), Flexible Fallback structure, Comprehensive loudspeaker management, Switchable filter functions, Definable equalization
- Interface: Colour TFT touchscreen 4.3"/480x272px, SCROLL/EDIT digital rotary encoder, mains power switch
- Dimensions: 2 RU x 19" x 512 mm / 2 RU x 19" x 20.2"
