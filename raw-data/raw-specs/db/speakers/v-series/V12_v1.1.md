# V12 (d&b audiotechnik) — Master Schema

**Category**: speakers | **Brand**: d&b audiotechnik (db) | **Schema Phase**: Phase 4 (d&b 13번째 제품군 투입, V8과 동시)

**스켈레톤 근거**: `speakers/db/V8_v1.0.md`(형제 파일)와 동일 원본(공용 OM, 개별 AE)에서 독립 확인. OM 자체 서술 "The V12 line array module is acoustically and mechanically compatible with the V8 and provides a 120° horizontal dispersion." — 두 제품은 수평 지향각(80°/120°)과 그에 종속되는 Max_SPL 수치만 다르고 나머지는 완전 동일(GSL8/GSL12, CCL8/CCL12 등과 동일한 "숫자=지향각" 명명 관례 재확인).
