# SB18 (L-Acoustics) — Master Schema

**Category**: speakers | **Brand**: L-Acoustics (LA) | **Schema Phase**: Phase 3-1(동일 브랜드 이기종 서브우퍼 추가 — Kara II/K1/K2/K3 등의 delay_defaults가 이미 참조하던 "SB18"의 실제 원본 최초 투입)

**스켈레톤 근거**: `speakers/LA/KS28_v1.5.md`를 뼈대로 사용 — SB18은 KS28과 동일한 LF 전용(Way_Count=1) 패시브 서브우퍼 아키텍처이며 섹션 구성이 동일하다. 다만 KS28과 달리 (1) 단일 18" 드라이버(KS28은 2x18"), (2) 8 Ohm 공칭 임피던스(KS28은 4 Ohm), (3) IN 전용이 아닌 IN+LINK 양쪽 4-point speakON 커넥터 구조(KS28은 IN 하나만 스펙에 명시)를 가진다.

**"SB18/SB18i/SB18m" 명칭 범위 확정**: 업로드된 `SB18_UM_EN.pdf`(버전 6.0)는 표지에 "SB18 (i/m)"이라 표기되어 있으나, 본문에 "In this document, the SB18 term and illustration will refer equally to SB18, SB18i or SB18m... The rigging system of each version has been designed to be compatible with a distinct main system: SB18 with Kara (II), SB18i with Karai, and SB18m with ARCS Focus/Wide."라는 명시적 조항이 있다. 이번 작업 지시 대상인 "SB18"은 이 3자 중 Kara (II)와 짝을 이루는 투어용 표준 버전이며, `Kara_II_v1.6.md`의 delay_defaults/preset_guide_and_matching이 "SB18"(접미사 없음)을 참조하는 것과 정확히 일치한다 — 따라서 이 파일은 UM Appendix C의 "SB18" 표를 채택했다. SB18i(소문자 i, "Karai"와 짝)와 별도 제품인 "SB18 IIi"(대문자 II, Kara IIi와 짝)는 완전히 별도의 OM(terminal block 커넥터, 48kg, IP55)을 가진 신세대 install 전용 제품이다.

### 주석 및 출처 (Notes & Sources)

**출처**: SB18_AE_EN.docx "Technical requirements"; SB18_UM_EN_6.0.pdf p.8-9("Loudspeaker configurations"), p.15(Appendix C spec table).
**프리셋 귀속**: AE는 "Usable bandwidth: 32-110Hz(-10dB), 34-100Hz(-6dB), 37-82Hz(-3dB)"를 단일 값으로 제공하고, UM Appendix C는 이 하한값을 [SB18_100] 프리셋 기준이라 밝힌다. [SB18_60](60Hz 상한) 프리셋은 정확한 -10/6/3dB 임계값별 수치가 원본에 없어 미확인.

**DSP_Preset_Name**: UM Appendix A(p.14)에서 확인된 표준(SB18_60/SB18_100) + 카디오이드(SB18_60_C/_Cx, SB18_100_C/_Cx) 프리셋 6종 전체 — Kara_II_v1.6.md의 delay_defaults가 참조하는 프리셋명과 완전히 일치함을 교차 확인.
