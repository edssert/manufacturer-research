# X8 (L-Acoustics) — Master Schema

**Category**: speakers | **Brand**: L-Acoustics (LA) | **Schema Phase**: Phase 3-1(이 카테고리 최초의 X 시리즈/점음원 제품)

**스켈레톤 근거**: X8은 Kiva II와 마찬가지로 **2-way 패시브 크로스오버**(단일 speakON 채널, 인클로저 내부에서 LF/HF 분리)이며 임피던스/파워 핸들링이 대역별이 아닌 통합값으로 보고된다. 이 카테고리 최초의 **점음원(point source)** 스피커이며 동축(coaxial) 드라이버 구조를 가진다. 준수 규격은 "EN 62368-1:2014"(K1~L2 계열의 "2006/42/EC Machinery Directive"와 상이).
