# ARCS Wide / ARCS Focus 프리셋 구성 (Constant Curvature WST systems)

> 원본 출처: `preset_guide_EN.pdf` p.67 (owner's manual EN v29.0)
> 상위 references 폴더(`../../preset-guide-en.pdf`)에 전체 원본 PDF 보관.

| Loudspeaker configuration | ARCS Wide/Focus | SB18 | Acoustic properties |
|---|---|---|---|
| line source | [ARCS_WIFO] | — | 55 Hz - 20 kHz |
| line source + SB18 | [ARCS_WIFO] | [SB18_60] | down to 32 Hz, reinforced LF contour |
| single enclosure | [ARCS_WIFO_FI] | — | 55 Hz - 20 kHz, flat response |
| single enclosure + SB18m | [ARCS_WIFO_FI] | [SB18_60] | down to 32 Hz, reinforced LF contour |

카디오이드 배열 시 `[SB18_60_C]`/`[SB18_60_Cx]` 사용.

출력 라우팅: `[ARCS_WIFO]`는 OUT1-4 전부 PA(IN A). `[ARCS_WIFO_FI]`는
OUT1/2=PA(IN A) + OUT3/4=PA(IN B).
