# A10 Wide/Focus 프리셋 구성 (Constant Curvature WST systems)

> 원본 출처: `preset_guide_EN.pdf` p.68-69 (owner's manual EN v29.0)
> 상위 references 폴더(`../../preset-guide-en.pdf`)에 전체 원본 PDF 보관.

A10 Wide/Focus와 A10i Wide/Focus는 동일 인클로저의 서로 다른 버전(원문 명시)
— 팩토리 프리셋과 권장 구성을 공유. KS21/KS21i도 마찬가지.

| Loudspeaker configuration | A10 Wide/Focus | KS21 | Acoustic properties |
|---|---|---|---|
| line source | [A10] | — | 66 Hz-20kHz(Focus)/67 Hz-20kHz(Wide) |
| line source + KS21 | [A10] | [KS21_100] | down to 31 Hz, reinforced LF contour |
| single enclosure | [A10_FI] | — | 66/67 Hz-20kHz, flat response |
| single enclosure (low latency) | [A10_MO] | — | 66/67 Hz-20kHz, flat response, low latency |
| single enclosure + KS21 | [A10_FI] | [KS21_100] | down to 31 Hz, reinforced LF contour |
| single enclosure + KS21 (low latency) | [A10_MO] | [KS21_100] | down to 31 Hz, reinforced LF contour, low latency |

카디오이드 배열 시 `[KS21_100_C]`/`[KS21_100_Cx]` 사용.

`[xx_MO]` 프리셋(레거시 XT/MTD 제외)은 앰프 컨트롤러의 저지연 동작 모드 사용.
서브우퍼 병용 시 서브우퍼도 저지연 모드 권장. 4채널 앰프에서는 저지연
채널셋+서브우퍼 채널셋을 조합한 커스텀 프리셋 생성 필요. 서브우퍼가 별도
4채널 앰프에서 팩토리 프리셋으로 구동되면 일반 지연 모드로 동작하므로,
`[xx_MO]` 저지연 채널에 추가 딜레이 필요: LA4/LA8 2.66ms, LA2Xi/LA4X/LA12X
3.00ms. 16채널 앰프에서는 서브우퍼도 `[xx_MO]`를 선택해 저지연 혜택 적용.

출력 라우팅: `[A10]`은 OUT1-4 전부 PA(IN A). `[A10_FI]`/`[A10_MO]`는
OUT1/2=PA(IN A) + OUT3/4=PA(IN B).

**우리 프로젝트 데이터 대조**: `a-series.data.js`의 A10 Focus freqs(-3dB 75Hz,
-6dB 70Hz, -10dB 66Hz)와 A10 Wide freqs(-3dB 78Hz, -6dB 72Hz, -10dB 67Hz)는
이 표의 "66 Hz-20kHz(Focus)"/"67 Hz-20kHz(Wide)"(-10dB 기준으로 추정)와 일치.
