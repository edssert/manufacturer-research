# A15 Wide/Focus 프리셋 구성 (Constant Curvature WST systems)

> 원본 출처: `preset_guide_EN.pdf` p.70-71 (owner's manual EN v29.0)
> 상위 references 폴더(`../../preset-guide-en.pdf`)에 전체 원본 PDF 보관.

A15 Wide/Focus와 A15i Wide/Focus는 동일 인클로저의 서로 다른 버전(원문 명시)
— 팩토리 프리셋과 권장 구성을 공유. KS21/KS21i도 마찬가지.

| Loudspeaker configuration | A15 Wide/Focus | KS21 | Acoustic properties |
|---|---|---|---|
| line source | [A15] | — | 41 Hz-20kHz(Focus)/42 Hz-20kHz(Wide) |
| line source + KS21 | [A15] | [KS21_60] | down to 29 Hz, reinforced LF contour |
| single enclosure | [A15_FI] | — | 42/43 Hz-20kHz, flat response |
| single enclosure (low latency) | [A15_MO] | — | 42/43 Hz-20kHz, flat response, low latency |
| single enclosure + KS21 | [A15_FI] | [KS21_60] | down to 29 Hz, reinforced LF contour |
| single enclosure + KS21 (low latency) | [A15_MO] | [KS21_60] | down to 29 Hz, reinforced LF contour, low latency |

카디오이드 배열 시 `[KS21_60_C]`/`[KS21_60_Cx]` 사용.

`[xx_MO]` 프리셋 저지연 관련 안내는 A10.md와 동일(레거시 XT/MTD 제외 저지연
모드, 딜레이 보정값 등).

출력 라우팅: `[A15]`는 OUT1-4 전부 PA(IN A). `[A15_FI]`/`[A15_MO]`는
OUT1/2=PA(IN A) + OUT3/4=PA(IN B).

**우리 프로젝트 데이터 대조**: `a-series.data.js`의 A15 Focus freqs(-3dB 47Hz,
-6dB 44Hz, -10dB 41Hz)와 A15 Wide freqs(-3dB 52Hz, -6dB 47Hz, -10dB 42Hz)는
이 표의 "41 Hz-20kHz(Focus)"/"42 Hz-20kHz(Wide)"(-10dB 기준으로 추정)와 일치.
