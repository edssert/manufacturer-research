# LA4X (L-Acoustics, Amplified Controller)

- Type: Amplified Controller
- Channels: 4 (4x4)
- Power class: Class D
- Connectivity: Milan-AVB, AES/EBU, analog
- IP Rating: IP30
- Rack: 2U, Weight: 11.3 kg / 24.9 lb
- Operating temp: 0°C/32°F to +50°C/122°F
- Mains: 100-240V ±10%, 50-60Hz
- Power supply: Universal SMPS with PFC
- I/O: Analog 4ch, AES/EBU 4ch(2x AES3) with SRC, Milan-AVB 4ch from 1 stream(up to 8ch)
- Network redundancy: Milan-certified seamless redundancy
- Connections: 2x etherCON 1Gb/s I/O+control, 4x female 3-pin XLR(2x switching analog/AES3), 4x male 3-pin XLR links
- Output: 4ch, 4x SpeakON, min 4Ω
  - 4Ω: 4x1000W / 8Ω: 4x1000W
- DSP: SHARC 32-bit float, 96kHz internal, flexible 4x4 routing
  - EQ: 8 IIR + 4 FIR linear phase per channel
  - Preset library: Factory presets, memory 011-127, 10 user memories
  - Latency: standard 3.84ms / low 0.84ms
- Ecosystem: L-Acoustics remote control software, LA Network Manager / Q-SYS, Crestron, Control4, Savant, SNMP, Extron
- Features: Array morphing, Air absorption compensation, Internal IIR/FIR EQ, L-DRIVE protection, Loudspeaker management, Output delay 0-1000ms
- Protection: Mains — over/under voltage, over temp, overcurrent / Outputs — overcurrent, short circuit, over temp
- Interface: LCD and LED's, rotary encoder + power/mute keys
