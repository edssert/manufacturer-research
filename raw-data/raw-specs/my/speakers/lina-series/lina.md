# LINA (Meyer Sound, LINA Series)

단일 모델 (폭 변형 없음).

- Type: Line Array, 2-way active
- Low Driver: 2×6.5″ long-excursion cone (4Ω)
- HF Driver: 1×3″ compression driver via REM® manifold (8Ω)
- Max SPL: 138 dB
- Horizontal Coverage: 100°
- Frequency — Operating Range: 65Hz–18kHz
- Frequency — Phase ±45°: 100Hz–18kHz
- Weight: 19.5 kg (43 lb)
- Dimensions (W×H×D): 515 x 213 x 389 mm / 20.27 x 8.38 x 15.32 in
- Connectors — Analog: XLR 3-pin female in / male loop out (XLR 5-pin 옵션 시 RMS 겸용)
- Connectors — AC: powerCON 20 in/loop out, 90–265V AC 자동전압
- Connectors — Amp: 3ch Class D, 합산 1950W peak
- Connectors — RMS Network: 2선 twisted-pair(옵션), 앰프 상태 리포트
- Power: Self-powered — built-in 3-channel Class D amplification (1950W peak total)
- Linear Peak SPL (M-noise): 132 dB (19dB crest factor)
- Linear Peak SPL (Pink noise): 128 dB
- Linear Peak SPL (B-noise): 130 dB
- 비고: 가장 작은 Meyer Sound 라인어레이 중 하나. Vertical coverage varies with array length/configuration
