# LEOPARD (Meyer Sound, LEOPARD Series)

- Type: Line Array, 2-way active
- Low Driver: 2×9″ long-excursion cone (2Ω)
- HF Driver: 1×3″ compression driver via REM® manifold (4Ω)
- Max SPL: 142 dB
- Frequency — Operating Range: 55Hz–18kHz
- Frequency — Phase ±30°: 92Hz–18kHz
- Weight: 34 kg
- Dimensions (W×H×D): 684 x 282 x 550 mm / 26.93 x 11.11 x 21.66 in
- Connectors — Analog: XLR 5-pin female in / male loop out(RMS 겸용), XLR 3-pin도 가능(오디오 전용)
- Connectors — AC: powerCON 20 in/loop out, 90–265V AC 자동전압
- Connectors — Amp: 3ch open-loop Class D, 합산 3900W peak
- Connectors — RMS Network: 2선 twisted-pair, Galileo GALAXY로 앰프 상태 리포트
- Power: Self-powered — built-in 3-channel Class D amplification (3900W peak total)
- Horizontal Coverage: 110°
- Linear Peak SPL (M-noise): 133.5 dB (18dB crest)
- Linear Peak SPL (Pink): 130 dB
- Linear Peak SPL (B-noise): 134.5 dB
