# PANTHER-M (Meyer Sound, PANTHER Series)

- Type: Line Array, 2-way active
- Low Driver: 2×12″ (4Ω)
- HF Driver: 2×3″ compression driver (8Ω)
- Horizontal Coverage: 95°
- Max SPL: 150.5 dB
- Weight: 68 kg
- Dimensions (W×H×D): 969 x 377 x 565 mm / 38.15 x 14.84 x 22.24 in
- IP Rating: IP55 (Neutrik TOP 커넥터 체결 시)

- Connectors — AC: Neutrik powerCON TRUE1 TOP, 200–240V AC 50/60Hz
- Power: Self-powered — built-in DSP amplification, no external amp required
