# TIGRA-W (Meyer Sound, TIGRA Series)

- Type: Line Array, 2-way active
- Low Driver: 2×10″ long-excursion cone
- HF Driver: 1×4″ compression driver coupled to horn
- Weight: 54 kg
- Dimensions (W×H×D): 862 x 324 x 565 mm / 33.92 x 12.76 x 22.25 in
- IP Rating: IP55 (Neutrik powerCON TRUE1 커넥터 체결 시)
- Connectors — Analog: Neutrik XLR 3-pin TOP airtight female in / male loop out
- Connectors — Digital: 2x Neutrik RJ45 etherCON TOP airtight (Milan AVB)
- Connectors — AC: Neutrik powerCON TRUE1 NAC3PX-TRUE1, 100–240V AC 50/60Hz
- Power: Self-powered — built-in DSP amplification, no external amp required
- Watt (nominal): 550W / Burst: 1270W(<1s) / Idle: 50W
- 비고: Arrays of 6+ units functionally create a cardioid polar pattern at all SPL levels
- Horizontal Coverage: 110°
- Max SPL: 145 dB
- AES75 max linear (dBz): 121 (RMS +3dBV)
- AES75 max linear (dBz peak): 139.5
