# Raw Images Archive

사용자가 제공한 **제품 사진 원본 파일**을 그대로(또는 최소한의 파일명
정리만 해서) 보관하는 폴더입니다. `raw-data/raw-specs/`와 짝을 이루며,
목적도 동일합니다 — 나중에 사진을 다시 확인하거나 교체할 때 "원본이
무엇이었는지" 되짚어볼 수 있게 하는 것.

## 구조

```
raw-data/raw-images/
  <mk>/                          제조사 축약형 (la / db / my)
    speakers/
      <series>/                  시리즈 폴더 (raw-specs와 동일한 폴더명)
        <원본 파일명>.png/jpg      파일명은 원본 그대로 두거나 공백만
                                   하이픈으로 정리(내용 자체는 가공 X)
```

## 관례

- 사용자가 Upload 폴더 등으로 제공한 사진 원본은 여기에 그대로 복사
  보관한다. 배경 제거, 뱃지 크롭, webp 변환 등 **가공은 하지 않는다** —
  가공본은 `assets/img/speakers/<mk>/<series>/`에 별도로 둔다.
- 파일명은 원본 그대로 쓰는 것이 원칙이나, 공백이 포함된 경우 하이픈으로만
  정리해도 된다(예: `soka black-Photoroom.png` →
  `soka-black-Photoroom.png`). 이름 자체를 새로 짓지 않는다.
- 여러 제품이 함께 찍힌 합성 사진(예: Syva+Syva Low 스택 컷)은 관련된
  모든 모델의 폴더 각각에 저장하지 않고, 대표 시리즈 폴더에 1개만
  보관한다 — 실제 사이트 반영(`assets/img`)에서는 관련 카드마다 복사해
  써도 된다.
